# No-Mad Community Platform

A helpful website for Central Asian immigrants in the U.S.

### 🌍 Features

- Mosques map
- Legal help info
- Food places
- Job listings
- News and blog
- Contact form

---

### ⚙️ Built With

- React.js
- React Router
- CSS
- Figma (design)
- GitHub
- Domain: [no-mad.us](https://no-mad.us)

---

### 📂 Folder Structure

- `components/` – Navbar, Footer, etc.
- `pages/` – Home, News, Blog, About, Contact
- `App.js` – Main layout
- `index.js` – Entry point

---

### ▶️ How to Run

1. Open terminal
2. Type:

```bash
npm install
npm run dev
